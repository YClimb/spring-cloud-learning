#### 安装教程

注:操作系统 mac system

1.  下载 consul

    https://www.consul.io/downloads
    [Mac OS X 64-bit  consul_1.8.4_darwin_amd64.zip]
2.  解压consul,复制 sudo cp consul /usr/local/bin

3.  启动consul 
    
     consul agent -dev
     
4.  访问地址:http://127.0.0.1:8500/ui
